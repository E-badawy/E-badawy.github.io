Quick Start

  1. Upload CSV, TSV, XLSX, JSON, XML, or DOCX.
  2. Run assessment to generate quality score and issue summary.
  3. Use transform actions to clean duplicates and missing values.
  4. Run EDA and model checks for baseline recommendations.

Common API Errors

`UNSUPPORTED_TYPE`Upload a supported file format.
`FILE_TOO_LARGE`Reduce file size or increase server limit.
`AI_NOT_CONFIGURED`Set `OPENAI_API_KEY` on backend server.

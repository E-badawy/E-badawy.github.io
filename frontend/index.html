      FAQ
      
          
      Common Questions
      
        
      
        
      
          What file formats are supported?
          
      CSV, TSV, XLSX, JSON, XML, and DOCX are supported for ingestion and profiling.
      
        
      
        
      
          How is the Data Health Score calculated?
          
      
            The score uses weighted checks across missingness, duplicates, outliers, and schema
            drift to provide a single quality indicator.
          
      
        
      
        
      
          Can I run transformations before modeling?
          
      
            Yes. You can remove duplicates, apply missing-value strategies, preview transformed data,
            and then run EDA and model checks.
          
      
        
      
        
      
          Do AI insights require additional setup?
          
      
            Yes. Configure `OPENAI_API_KEY` on the backend server before using AI insight generation.

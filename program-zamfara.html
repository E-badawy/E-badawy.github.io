Program Overview

This bootcamp series was designed to strengthen data competencies through structured modules on analytical thinking, data preparation, visualization, and introductory machine learning.

The delivery model combined instructor-led sessions, guided practicals, and context-specific examples from public and development-sector workflows.

Activities & Value Delivered

Key activities included curriculum delivery, hands-on notebooks, participant mentoring, and competence checks tied to real-world decision use cases.

Value delivered includes improved data interpretation capacity, stronger reporting confidence, and reusable learning artifacts for continued practice.
